# Where Am I Looking (WAIL)

WAIL is a Fabric 1.21.10 mod which shows the exact decimal world coordinate of the rendered block model surface under your crosshair.

## Commands

`/wail`

Shows current settings.

`/wail on`

Turns the HUD on.

`/wail off`

Turns the HUD off.

`/wail decimalamount <0-10>`

Sets how many decimal places are shown.

`/wail decimaltype <0.1|0.0625>`

Sets whether coordinates use normal decimal-place rounding or nearest 1/16 block-pixel increments.

`/wail cursorsnapping`

Toggles cursor snapping to nearby 1/16 block-pixel corners.

`/wail cursorsnapping on`

Turns cursor snapping on.

`/wail cursorsnapping off`

Turns cursor snapping off.

`/wail cursorsnapping sensitivity [0.0-0.03125]`

Shows or sets how aggressively cursor snapping locks to nearby 1/16 block-pixel corners.

`/wail color <name|RRGGBB|#RRGGBB>`

Sets the HUD text color. Color names: `white`, `black`, `red`, `green`, `blue`, `yellow`, `cyan`, `magenta`, `gray`, `orange`.
